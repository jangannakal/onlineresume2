My Web-Dev Noob List.

After a series of mis-starts, replete with public declarations that coding was not for me, and that I would never contribute to the tyranny of the technocracy, here I am, at last, fumbling my way through what, I think, is called coding.

Enjoying the process as I move from manual HTML/CSS to utility-first CSS frameworks on my next site build.

Any developer is laughing at this list but I have to start somewhere.

Learning Responsive Web Design

Steep learning curve here as Clarissa smashed me through the responsive web lookinglass. My familiarity with CSS basics magnified exponentially thanks to this great book.

HTML and CSS: Design and Build Websites

Finally, a book that put it all together for me. Is it the layout? The big letters? The pictures? No idea but the ease of reference and simplicity of ordered subjects made some measure of control achievable.

Grid systems in graphic design

Loved the English German side-by-side columns and just about everything else covered. Gave me design goals to reach for.

Order in Space: A Design Source Book

One of these resources far beyond my ken but a joy to leaf through and ponder the concepts.

The Wall Street Journal Guide to Information Graphics

Bloomberg aesthetic. My deck building skills are nested in the 'modern cave man' school so lots of room for improvement. Best used for B2B business proposals and funding related requests.

Landing Page Optimization: The Definitive Guide to Testing and Tuning for Conversions

Pretty much all everything could want for all things SEO. A constant reference guide for me when thinking web strategy and not wading thru CSS or JS.

Eloquent JavaScript Third Edition

Clean and articulate writing that almost makes sense to this noob. A book I will read and re-read many times. Free version online - thank you, Marijn!

JavaScript and JQuery: Interactive Front-End Web Development

Following on the heels of Duckett's HTML and CSS book, this is my adult picture book guide to JS. Still working my way through it.

WordPress for Beginners 2021 : A Visual Step-by-Step Guide to Mastering WordPress

Followed everything up to where I wanted to customize my own WP theme. Gutenberg UX sucks so either I learn PHP and CSS better or abandon WP completely.
